# 📥 Inbox

The inbox is your **capture zone**—a temporary holding area for ideas, thoughts, and notes that haven't been processed yet.

## Purpose

- **Quick capture**: Write ideas down immediately without worrying about organization
- **Processing buffer**: A staging area before notes move to their permanent home
- **Low friction**: Reduce barriers to capturing thoughts

## When to Use

✅ **Use the inbox for:**
- Quick ideas that pop into your head
- Interesting quotes or snippets you want to save
- Meeting notes that need to be processed
- Links to articles you want to read and notes about
- Half-formed thoughts that need refinement
- Screenshots, images, or resources to process later

❌ **Don't use the inbox for:**
- Daily notes (use `01-Fleeting/Daily/` instead)
- Already-refined permanent notes (put directly in `03-Permanent/`)
- Project work (use `05-Projects/` instead)

## The Inbox Rule

> **Keep your inbox under 20 items**

When your inbox grows beyond 20 items:
- Schedule time to process it
- Move notes to their appropriate homes
- Delete what's no longer relevant

## Processing Workflow

Process your inbox **weekly** (or when it exceeds 20 items):

### Step 1: Review Each Note

For each item in the inbox, ask:
1. **Is this still relevant?** If no → Delete or archive
2. **What type of note is this?**
   - Quick thought → Convert to permanent note or discard
   - Source material → Move to `02-Literature/`
   - Refined idea → Move to `03-Permanent/`
   - Project-related → Move to `05-Projects/`
   - Reference material → Move to `07-Resources/`

### Step 2: Transform & Move

- **Fleeting → Permanent**: Refine the idea, write in your own words, add links
- **Literature**: Structure with the literature note template
- **Resource**: Add to appropriate resource subfolder
- **Project**: Link to or create project note

### Step 3: Create Connections

- Link new notes to existing notes
- Update relevant MOCs
- Add backlinks where appropriate

### Step 4: Clean Up

- Delete processed notes from inbox
- Archive anything you're keeping "just in case"

## Tips for Effective Inbox Use

### Capture Immediately
When an idea strikes:
1. Create a new note in inbox
2. Write the core idea (even if rough)
3. Add context about where it came from
4. Move on—don't perfect it now

### Use Templates
Even in the inbox, using a basic structure helps:
```markdown
# Quick Capture

**Idea**: [Main thought]

**Context**: [Where did this come from?]

**Next**: [What should happen with this?]
- [ ] Process into permanent note
- [ ] Find related notes
- [ ] Research more
```

### Time-box Processing
- Set a timer for 30 minutes
- Process as many notes as possible
- Don't aim for zero inbox—aim for useful notes

### Review Before Deleting
- If unsure, move to `08-Archive/` instead of deleting
- You can always retrieve it later
- Better to be lean than hoard

## Quick Capture Methods

### Obsidian Quick Switcher
1. Press `Cmd/Ctrl + O`
2. Type new note name
3. Start writing immediately

### Daily Note Reference
If you prefer capturing in daily notes:
1. Write in today's daily note
2. During weekly review, extract to inbox
3. Then process from inbox to permanent

### Mobile Capture
- Use Obsidian mobile app
- Enable "Default location for new notes" → `00-Inbox/`
- Voice notes, quick photos → process on desktop

## Inbox Health Indicators

| Status | Inbox Count | Action |
|--------|-------------|--------|
| 🟢 Healthy | 0-10 items | Keep capturing! |
| 🟡 Attention | 11-20 items | Schedule processing session |
| 🔴 Urgent | 21+ items | Process immediately |

## Common Mistakes

❌ **Perfectionism**: Spending 30 minutes formatting an inbox note
- ✅ Instead: Quick capture, refine during processing

❌ **Inbox zero obsession**: Processing items before you're ready
- ✅ Instead: Let ideas marinate, process when you have time

❌ **Using inbox as storage**: Leaving items for months
- ✅ Instead: Process weekly, keep it flowing

❌ **Avoiding difficult notes**: Skipping items that need more thought
- ✅ Instead: Move hard items to dedicated project notes

## Example Inbox Note

```markdown
# Compound Interest in Knowledge

Quick thought: Knowledge compounds like interest—small daily learning
creates exponential growth over time.

Source: Conversation with mentor, 2026-04-21

Related ideas:
- Spaced repetition
- Zettelkasten method
- Atomic habits

Next steps:
- [ ] Research compound learning studies
- [ ] Create permanent note
- [ ] Link to learning MOC
```

After processing → becomes permanent note in `03-Permanent/`
